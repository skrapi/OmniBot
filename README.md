# OmniBot
An idea for an omnidirectional robot

## Dimensions
Circular robot 
* 10 - 20 cm diameter
* 5 cm high
* All enclosed

## Specifications
* Run off a LiPo
* Quiet
* 3 DC brushed motors with encoders
* Omnidirectional wheels
* RF control
* IMU position control
* Wireless Charging
